OUR PORTFOLIO

Completed projects for various
industries.

  • AnimaManager

    AnimaManager - a desktop application developed for online render service AnimaRender farm. The app interacts with AnimaRender backend. AnimaManager allows to start render directly from user’s modelling software, monitor render jobs and perform all other actions as one can do on the website (calculate the cost of the project, recharge his/her account, read news of the service etc.) The app can serve as a full-scale substitute of the website with one significant advantage: synchronization. This function allows to synchronize all the results of render jobs in real time to the specified folder on the user’s computer. The app can be used for submitting render jobs either with built in plugins for various modeling software or by itself. AnimaManager is available both for Mac OS and Windows.

    Project tech stack: ElectronJS, Sequelize, ReactJS, Redux, Javascript

  • AnimaPlugins

    AnimaPlugins are parts of complex development for online render service Animarender.com. The plugins are utilities that are built into specific 3D modelling software, e.g. 3ds Max, Cinema 4D, Blender etc, allowing to collect all the assets used in user’s 3D project, prepare it in the proper way and submit for render on the AnimaRender farm in one click. On top of this, the AnimaPlugins can detect errors that can lead to the render failure and either fix it or offer the ways of fixing to the user. The plugins are integrated in AnimaManager app and can be connected to the user’s software by the first launch of AnimaManager and automated detection of all the 3D software packages installed on the computer.

    Project tech stack: C++, Python

  • MFO

    ManyMoney.ru is a Russian online personal loan service, which allows Russian citizens to get loan credited to their credit card within 10 minutes when applying only with passport information and card number. This modern and user friendly service was developed in accordance with a law of Russian Federation to meet all requirements of Central Bank. Loan system works under financial license and orchestrates the work of identification services, information exchange with credit history and external scoring services, processing unit and integrated payment system.

    Project tech stack: Tornado, Pewee, aio-pika, ReactJS, Python, Javascript

  • Arsenal

    Arsenal is a personal account for the website of Arsenal, a guard service company located in Russia. Using the personal account clients of Arsenal can monitor the state of their properties being guarded by the company in real time (a house, an apartment, office premises etc.) The personal account also allows to pay for the security provider services and control the current account balance.

    Project tech stack: Django, ReactJS, Python, Javascript

  • CG - Doo

    CG -Doo is an online service directly connecting 3d artists and studios with producers, advertising agencies. Any freelancer or visual effects studio can get direct orders without intermediaries. The service allows to run the complete cycle of 3d graphics creation from technical description of the job, through splitting complex tasks into stages and milestones to contract work and money transaction. The system has built in rating system which is based on customer feedback and skills evaluation which allows to guarantee a perfect match with best pricing for any task posted.

    Project tech stack: ReactJS, Redux, Django, Python, Javascript

  • Tralrf.ru

    Tralrf.ru is a web service which allows to search and order any construction equipment spare parts all over Russia. During several years of support we have significantly reworked the logic, structure and connections as the business focus have changed. We have connected a bunch of suppliers and now service allows to find and order any spare parts even those which were never shipped to Russia through official dealers.

    Project tech stack: Yii2, Django, Vue.js, Python, PHP, Javascript

  • Anima ML

    Anima ML - online machine learning platform for data processing and deep learning based on private data center of Animarender company. The platform allows to use almost unlimited computing power for ML tasks which scales seamlessly and instantly if needed. All users can develop and process models with all potential of modern machine learning science including gradient boosting and deep neural networks. -

    Project tech stack: pandas, numpy, eli5, xgboost, catboost, sklearn, matplotlib, keras, fastai, Python

  • Sovetnik Archive House

    Sovetnik Archive House is an online service aimed to simplify the navigation and management of the cells of the Archive House (storage of archives). Our team wrote a code that allows real-time monitoring of the state of each archive cell: the presence of documents in it, actions performed with documents, the owner of the cell, etc. The service allows the staff of the Sovetnik Archive House to easily navigate among lots of racks and cells, which simplifies subsequent work with the clients.

    Project tech stack: Zend Framework, Flask, SQLAlchemy, Android SDK, Java, Python, Javascript

  • AnimaCore

    Animarender Core is a part of complex development for online rendering service, so-called render farm. The core is based on docker containers and custom micro services such as synchronization, billing, task management and backend engine. Altogether their work allows asynchronous processing of hundreds and thousands of requests per second, continuous exchange with thousands of files between datacenter and unlimited amount of users worldwide simultaneously.

    Project tech stack: Tornado, Pewee, Nodejs, Typescript, Socketio, Mongoose, Python, Javascript, C#
